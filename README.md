# Design Portfolio

Manage your design portfolio and inspiration in Obsidian: case studies, project showcases, mood boards, and client work logs.

## Features

- **Case Studies** — Create structured case study notes with Brief, My Role, Process, Outcome, Learnings, and Assets sections
- **Inspiration Capture** — Quickly log inspiration with source URL, tags, and notes
- **Mood Boards** — Generate mood board notes with an image grid template
- **Portfolio Sidebar** — Browse case studies by year and inspiration by tag in a dedicated sidebar panel

## Usage

### Commands
- `Design Portfolio: New case study` — Opens the case study creation modal
- `Design Portfolio: New inspiration` — Opens the inspiration capture modal
- `Design Portfolio: New mood board` — Creates a new mood board note
- `Design Portfolio: Open portfolio` — Opens the portfolio sidebar

### Settings
- **Portfolio folder** — Where case study notes are saved (default: `Portfolio`)
- **Inspiration folder** — Where inspiration notes are saved (default: `Portfolio/Inspiration`)
